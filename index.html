  • Import and Export Contacts

    Import and Export Contacts

    Managing and organizing contacts in Android phones is quite easy and provided with various options.

     

    If you are still managing your contacts manually here is the solution. We will show you how to Copy Contacts to Google Account or Import Contacts to SIM card. Follow these steps:

    Copy Contacts to Google Account.

    1. Select Settings.  

     

    2. Select Contacts.  

     

    3. Select SIM card contacts.

     

    4. Select SIM 1 or SIM 2.

     

    5. Select Export from SIM Card.  

     

    6. Select Contact(s) to copy.  

      Note: To select all contacts in one go, click on Select All.

     

    7. Select Export from SIM card.  

     

    Import Contacts to SIM card.

    1. Select Settings.  

     

    2. Select Contacts.  

     

    3. Select Import/Export Contacts.

     

    4. Select SIM 1 or SIM 2.

     

    5. Select Import to SIM card.  

     

    6. Select Contact(s) to copy.  

      Note: To select all contacts in one go, click on Select All.

     

    7. Select OK.  

     

    8. Select Import.
